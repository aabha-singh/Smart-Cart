# 🛒 SmartCart - React E-commerce UI

## 🚀 Project Overview
SmartCart is a modern e-commerce web application built using React. It allows users to browse products, search items, apply filters, and manage a shopping cart with a smooth and responsive user experience.

---

## ✨ Features
- 🔍 Product Search functionality
- 🎯 Category-based filtering
- 🛒 Add to Cart functionality
- 📦 Dynamic product listing using API
- ⚡ Responsive UI using Tailwind CSS
- 🔄 State management using React Hooks

---

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Axios

---

## ⚙️ Setup Instructions

### 1. Clone the repository
```bash
git clone https://github.com/aabha-singh/smart-cart.git
```

### 2. Navigate to project folder
```bash
cd smart-cart
```

### 3. Install dependencies
```bash
npm install
```

### 4. Run the project
```bash
npm run dev
```

## 💡 Approach
- Reusable components like Navbar, ProductCard, FilterBar banaye
- `useState` se state manage kiya (cart, search, filter)
- `useEffect` se API data fetch kiya
- Clean UI aur user-friendly design pe focus kiya

## 📌 Assumptions

- Cart frontend-based hai (no backend)
- Basic validations implemented
